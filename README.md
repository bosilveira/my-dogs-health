## 🚀 Techs

Neste Projeto usamos as seguintes tecnologias:

- [x] ReactJS
- [x] Redux
- [x] React Router Dom
- [x] Yup
- [x] Chakra UI


### Dependencies

- [x] Yarn
- [x] Axios
- [x] JSON-Server

### Features

- [x] Auntenticação Login/Senha
- [x] Possibilidade de Cadastro de cãezinhos na plataforma
- [x] Cadastrar ação para cada cão em particular (vacina,medicamento..)
- [x] Perfil Com dados do tutor e do cão
- [x] Perfil lateral com dados do usuário logado
- [x] Rota individual para cada cão com detalhes mais especificos 


## 💻 O projeto

Este projeto foi Desenvolvido durante o Capstone na Kenzie Academy Brasil, neste projeto utilizamos de dados estatiscos fornecidos por consulta na internet para identificar uma dificuldade das pessoas em conseguir manipular todo histórico de seu animal, então criamos o MydogsHealth que é uma plataforma onde você pode ter total controle e atividade de seu cãozinho, como vacinas, sáude e tudo mais.

Neste projeto utilizados metodologia Scrum assim como Gitflow para orquestrar nosso repositório.

### Para rodar o projeto em sua máquina

- Necessário : [Node.js](https://nodejs.org/en/download/) e  [Yarn](https://yarnpkg.com/) para executar o projeto.

**Clone o projeto deste repositório**

```bash
$ git clone git@github.com:bosilveira/my-dogs-health.git
```

```bash
$ cd my-dogs-health
```

**Siga esses passos no terminal**

```bash
# Instale as dependencias 
$ yarn


# Inicie a aplicação
$ yarn start
```
